Welcome to NASA downloader!


about the app:

Get the NASA official picture of the day!
Each day a different image or photograph of our fascinating universe is featured,
along with a brief explanation written by a professional astronomer.
You can also browse for previous images at a specific date.


How to run the app:

built on python 3.8.2
There is a requirements.txt file, import the modules.
run on your Python environment the main file: "nasa_main.py".
